# Industrial Machine Downtime Tracker (Python + SQL Server ORM)

### 📌 Project Overview
This project simulates an Industry 4.0 data pipeline. It automatically captures machine downtime events from an industrial shop floor (simulating PLC tags) and logs them into a relational database using Python and SQLAlchemy ORM.

### 🛠️ Tech Stack
* **Language:** Python 3.x
* **Database Object Mapping:** SQLAlchemy ORM
* **Database:** SQLite (Default for demonstration, easily scalable to Microsoft SQL Server)

### 🚀 Business Value
In modern manufacturing, tracking Overall Equipment Effectiveness (OEE) and Mean Time To Repair (MTTR) is critical. This script automates data ingestion, avoiding manual paper logs and providing clean data for Power BI dashboards or AI predictive maintenance analysis.
